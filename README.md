# Pokémon Veyra

RPG fã retrô/top-down criado por **Luiz Matheus** para estudo, portfólio individual e experimentação com jogos web em HTML, CSS e JavaScript puro.

## Versão

**v1.5 — Top-Down Crystal Rebuild**

## Destaques

- Exploração top-down em tiles, com cidades, rotas, interiores, gruta, costa e ginásio.
- Introdução jogável com quarto, mãe, laboratório e escolha de inicial.
- Batalha em estilo clássico com nome, level, HP, EXP, sprites, Poké Balls da equipe e animações.
- Captura, equipe, box, Pokédex, bolsa, loja, treinador rival, ginásio, experiência, evolução e status.
- Música gerada por código com variações por local: casa, laboratório, cidade, rota, floresta, gruta, costa, ginásio e batalha.
- Save local no navegador.
- PWA instalável como atalho/app em celulares compatíveis.

## Controles

- Setas / WASD: mover.
- Enter / Z / botão A: interagir.
- Esc / X / botão B: menu.
